# Infrastructure Behavioral Taxonomy

Behavioral taxonomy defines formal classes of infrastructure behavior:
- COMPENSATORY_SYSTEM
- OSCILLATORY_SYSTEM
- FRAGMENTATION_PRONE_SYSTEM
- DELAYED_RESPONSE_SYSTEM
- PROPAGATION_SENSITIVE_SYSTEM
- CONVERGENCE_RESISTANT_SYSTEM
- COMPRESSION_DOMINATED_SYSTEM
- RECOVERY_CAPABLE_SYSTEM
- ENVIRONMENTALLY_COUPLED_SYSTEM

Each class requires topology signatures, propagation tendencies, recovery characteristics, and replay-linked evidence requirements.
