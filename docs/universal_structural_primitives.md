# Universal Structural Primitives

Universal structural primitives define the foundational language of infrastructure behavior across domains.

Primitives include propagation, convergence, compensation, fragmentation, stabilization, synchronization, latent pressure, oscillation, compression, release, persistence, recovery, divergence, coupling, and decoupling.

Each primitive is defined with structural meaning, topology signature, temporal behavior, evidence requirements, replay indicators, compatible archetypes, and operator-facing explanation language.
