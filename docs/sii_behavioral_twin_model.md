# SII Behavioral Twin Model

Neraium behavioral twins are structural cognition models, not generic physics simulators.

## Required Twin Capabilities

- replay history overlays
- continuation pathway overlays
- propagation simulation overlays
- convergence simulation overlays
- ontology-linked cognition primitives
- subsystem interaction memory
- operational timing evolution
- evidence replay lineage views
- historical cognition comparisons

## Twin Role in SII

Behavioral twins serve as replayable structural intelligence artifacts for operator reasoning, validation, and audit defensibility.
