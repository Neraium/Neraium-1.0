# Foundational Reasoning Substrate

The foundational reasoning substrate unifies primitives, ontology, replay, evidence lineage, topology context, structural memory, simulation context, and behavioral twin context.

Every reasoning output must reference:
- evidence lineage basis
- replay support
- ontology context
- topology context
- structural memory context
- confidence and uncertainty basis

The substrate produces explainable operator-facing reasoning bundles and explicitly avoids black-box claims.
