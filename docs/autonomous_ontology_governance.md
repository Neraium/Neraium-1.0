# Autonomous Ontology Governance

Autonomous ontology governance allows the platform to propose ontology improvements while preserving human validation authority.

Governance can propose new primitives, detect weak archetypes, identify conflicts, flag duplicate concepts, and report replay validation gaps.

Promotion states:
- PROPOSED
- NEEDS_EVIDENCE
- UNDER_REVIEW
- VALIDATED
- REJECTED
- DEPRECATED

No ontology proposal is auto-promoted without evidence and operator review.
