# SII Evidence Lineage Standard

All cognition outputs must be traceable through evidence lineage.

## Required Lineage Dimensions

- contributing signal references
- topology evidence references
- propagation confirmation references
- persistence evidence references
- historical memory references
- replay frame references

## Traceability Requirements

- lineages are timestamped and replay-linked
- lineages are queryable from audit endpoints
- lineages preserve continuation reasoning context
