# SII Reference Architecture

Neraium implements Systemic Infrastructure Intelligence (SII) as a replayable, evidence-driven operational cognition architecture.

## Canonical Layers

1. Telemetry Normalization Layer
2. Structural State Layer
3. Topology Cognition Layer
4. Propagation Reasoning Layer
5. Structural Memory Layer
6. Archetype Ontology Layer
7. Evidence Lineage Layer
8. Replay/Audit Layer
9. Continuation Modeling Layer
10. Operator Cognition Interface Layer

## Distinguishing Properties

- Explainable and replay-first cognition artifacts
- Evidence lineage and audit traceability by default
- Continuation and convergence modeled as operational ranges
- Non-actuating and operator-centric interpretation
- Structural cognition language preserved across backend, APIs, and UI
