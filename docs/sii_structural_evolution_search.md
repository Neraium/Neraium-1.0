# SII Structural Evolution Search

Structural evolution search targets replay-backed behavior, not raw telemetry retrieval.

## Search domains
- Persistent graph memory
- Replay frames
- Archetype emergence
- Propagation pathways
- Convergence and recovery histories
- Evidence lineage references

## Example query intent
- Find convergence failures with delayed recovery
- Find propagation patterns involving airflow and thermal lag
- Find structural compression before fragmentation

## Output model
Search results provide behavior matches, evidence context, and replay references.
