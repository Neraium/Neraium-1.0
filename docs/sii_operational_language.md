# SII Operational Language

Neraium standardizes a fixed operational vocabulary for Systemic Infrastructure Intelligence.

## Canonical Terms

- topology drift
- propagation pathway
- continuation window
- subsystem fragmentation
- structural convergence
- structural compression
- archetype emergence
- cognition state
- structural memory
- operational timing
- topology coherence
- evidence lineage
- replay fidelity
- behavioral twin
- structural primitive

## Forbidden Terms

- predicted failure
- AI recommendation
- anomaly score
- remaining useful life
- automated action
- bot/copilot
- magic insight
- generic alert
