- propagation pathway
- continuation window
- subsystem fragmentation
- structural convergence

- archetype emergence
- cognition state
- structural memory
- operational timing

- evidence lineage
- replay fidelity

- AI recommendation
- anomaly score
- remaining useful life
- automated action
- bot/copilot
- magic insight
