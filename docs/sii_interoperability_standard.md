# SII Interoperability Standard

SII interoperability provides graph-friendly exchange contracts for structural cognition data.

## Interoperability artifacts
- `SIIEvent`
- `SIIContextEntity`
- `SIIContextRelationship`
- `SIIReplayFrameExport`
- `SIIEvidenceExport`
- `SIIOntologyExport`

## Exchange model
The model uses entity-property-relationship semantics compatible with graph/context approaches while preserving SII language:
- topology cognition
- propagation pathways
- continuation windows
- structural memory
- evidence lineage
- replay fidelity

## Scope boundaries
- Read-only ingestion and export only.
- No control-plane schema for command dispatch.
- No actuation intent fields.
