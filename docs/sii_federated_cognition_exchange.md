# SII Federated Cognition Exchange

Federated exchange allows facilities to share cognition primitives without exposing sensitive telemetry.

## Exchange scope
- Archetype signatures
- Topology state summaries
- Propagation primitives
- Convergence patterns
- Replay fingerprints
- Evidence summaries
- Ontology extensions

## Prohibited exchange
- Raw telemetry
- Customer-sensitive operating identifiers
- Control commands and actuation payloads

## Privacy profile
Payloads use privacy-preserving summaries and hashed facility identifiers.
